Список сокращений
=====================

#. **АНПП** - архитектурно-независимое параллельное программирование
#. **ПВС** - параллельная вычислительная система
#. **ПЛИС** - программируемые логические интегральные схемы
#. **СТМФППВ** - статически типизированная модель функционально-потоковых параллельных вычислений
#. **ФПМПВ** - функционально-потоковая модель параллельных вычислений
#. **ФПП** - фунционально-потоковый параллельный
#. **ФППП** - функционально-потоковое параллельное программирование
#. **ФПППП** - функционально-потоковая парадигма параллельного программмирования
#. **ЯФППП** - язык функционально-потокового параллельного программирования
